## Kovid-MY

Kovid-MY is a Discord Bot that reports Malaysia COVID-19 Stats from data provided by MoH Malaysia

Data is retrieved from [COVID-19 MoH Public Repo](https://github.com/MoH-Malaysia/covid19-public)
